# UltimateVLESS-Zapret
Ультимативное решение представляющая собой, спектр упрощения пользования интернетом людям с СНГ.
 1. Конфигуратор Vless + Добавление ключей (80%).
 2. Автоматизация использования Vless ключей, с наиболее валидные (Скоро).
 3. Добавления запрета, с визуальным интерфейсом, для упрощения пользования (Скоро).
 4. Минимизация интерфейса - простота использования!
